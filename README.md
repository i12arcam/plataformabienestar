# MindAura (TFG)

Frontend desarrollado como Trabajo de Fin de Grado.

## Descripción

Aplicación móvil orientada a que los usuarios puedan gestionar su salud mental y emocional, mediante seguimiento del estado de ánimo, técnicas de mindfulness, mensajes motivacionales, consejos, actividades y 
metas personales.

## Documentación

La documentación completa del Trabajo de Fin de Grado está disponible en la carpeta /docs.

## Tecnologías

- Kotlin + Jetpack Compose
- MPAndroidChart para gráficas
- YouTube Player API (pierfrancescosoffritti)
- Retrofit 2 + Gson para consumo de API REST

## Cómo iniciar

Debe estar el backend funcionando. Si no, la aplicación arrancará pero no funcionará.

Para arrancarla, debe instalarse Android Studio, y ejecutarla desde ahí.
